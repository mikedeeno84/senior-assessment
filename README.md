## Extra Credit

For the brave, the extra credit (implementing Promise.map) is available in `tests/extra-credit`. Run with `npm run extra-credit`.

## Getting started

Fork and clone this repository. Then...

```bash
npm install
```

To run server tests:

```bash
npm test
```

To run browser tests:

```bash
npm run browser-test
```
